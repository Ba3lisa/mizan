# Contributing to Mizan

Mizan is an open-source civic transparency platform. While the codebase is open for review and forking, contributions are currently managed by the maintainer.

## Reporting Data Errors

Found incorrect data? Please open a GitHub Issue using the "Data Correction" template. Include:
- The specific number that's wrong
- The correct value with a source URL proving it
- The page where you found the error

## Code Contributions

Code contributions are not currently accepted via pull requests. If you'd like to suggest a feature or improvement:
1. Open a GitHub Issue with the "Feature Request" label
2. Describe the feature and why it would benefit users

## Data Sources

All data on Mizan comes from official Egyptian government publications and accredited international organizations. See [/methodology](/methodology) for details.
